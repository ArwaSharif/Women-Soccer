# Introduction

The Women's Soccer application is an interactive app that introduces information about different women's national teams.

### Technologies Used

This is full-stack MERN application that utilized technologies including:
- CRUD operations
- AJAX calls
- date-fns library,
- concurrently,
- react-router-dom,
- dotenv,
- react-bootstrap.
